2. activate it

    $pip install -r requirements.txt

4. go 'pytest_operations'
5. if want to run all tests:
    $pytest -v

    $cd dir-basic
    $pytest -v
7. if see 'passed' like 'folder/file-name::test_function passed'
it is working
